DownLoadManager
===============

介绍：     基于ASIHTTP 的多线程下载管理功能模块，支持异步下载、暂停和恢复下载等等功能。暂停某个下载任务，再次恢复该下载任务时，不会重头下载，而是在已下载的地方开始下载（即断点下载）。具体有如下特色： 

    1. 完整的UI设计，可直接拿来使用； 

    2. 异步，后台，多线程（ASI提供）； 

    3. 最大同时下载数设置； 

    4. 下载任务排队，暂停，等待，继续，删除某个下载任务； 

    5. 记录下载的未完成和已完成任务，并在下次启动时加载；
